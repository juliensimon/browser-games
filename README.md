# Browser Games

Classic arcade games recreated in the browser with vanilla JavaScript and HTML5 Canvas. No frameworks, no build tools, no dependencies — just open and play.

## Games

| Game | Description | Play |
|------|-------------|------|
| [Asteroids](asteroids/) | The 1979 Atari classic. Pilot your ship through an asteroid field, blast rocks into fragments, and dodge UFOs. Vector-style graphics with phosphor glow effects. | Open `asteroids/index.html` |

## How to Run

Clone the repo and open any game's `index.html` in a browser:

```bash
git clone https://github.com/juliensimon/browser-games.git
open browser-games/asteroids/index.html
```

No server required — everything runs client-side.
